# Lost&Foud

A web-based Lost and Found Management System for students and staff.

## Features
- Report Lost Items
- Report Found Items
- Browse Items
- Search & Filter
- Item Details
- Admin Dashboard

## Tech Stack
- React
- Vite
- Tailwind CSS
- Node.js (planned)
- MySQL (planned)
